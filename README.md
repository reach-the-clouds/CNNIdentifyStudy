# 计算机视觉项目集合

这个仓库包含了多个计算机视觉相关的项目，涵盖了图像分类、数字识别、垃圾分类和水果识别等多个应用领域。

## 项目结构

### 1. 垃圾分类 (Garbage classification)

一个基于CNN的垃圾分类识别项目，能够识别不同类型的垃圾。

**主要功能：**
- 数据集处理和增强
- 自定义数据集创建
- CNN模型训练
- 可视化界面展示

**核心文件：**
- `5.训练垃圾分类识别cnn模型 (1).py` - CNN模型训练脚本
- `6.tk实现可视化.py` - 基于tkinter的可视化界面
- `cnn.py` - CNN模型定义
- `garbagedataset.py` - 自定义数据集类
- `server.py` - 后端服务器实现

### 2. 线性回归 (Linear)

包含线性回归相关的数学基础和实现，以及二分类和三分类问题的解决方案。

**核心文件：**
- `4.前向传播与损失函数.py` - 前向传播和损失函数实现
- `5.反向传播与学习率.py` - 反向传播算法实现
- `8.基于pytorch框架的线性回归-简化.py` - 基于PyTorch的线性回归实现
- `二分类.py` - 二分类问题解决方案
- `三分类.py` - 三分类问题解决方案

### 3. 数字识别 (numerals_recognition)

基于MNIST数据集的数字识别项目，包含FNN和CNN两种模型实现。

**核心文件：**
- `7.模型训练.py` - FNN模型训练
- `10.cnn模型训练.py` - CNN模型训练
- `12.tk数字识别.py` - 基于tkinter的数字识别界面
- `CNN.py` - CNN模型定义
- `fnn.py` - FNN模型定义

### 4. ResNet18模型应用 (resnet18)

基于预训练的ResNet18模型进行图像分类任务，包含模型修改、训练和评估。

**核心文件：**
- `1.模型查看和修改.py` - ResNet18模型结构查看和修改
- `2.模型训练和评估.py` - 模型训练和评估流程
- `3.测试模型.py` - 模型测试脚本

### 5. 水果识别 (水果识别python代码)

一个水果分类识别项目，能够识别多种水果类型。

**主要功能：**
- 数据集处理和增强
- 自定义数据集创建
- CNN模型训练
- Web界面展示

**核心文件：**
- `5.训练水果分类识别cnn模型.py` - CNN模型训练脚本
- `cnn_fruits.py` - CNN模型定义
- `fruitsdataset.py` - 自定义数据集类
- `server.py` - 后端服务器实现

## 环境要求

- Python 3.7+
- PyTorch
- torchvision
- numpy
- PIL (Pillow)
- tkinter (用于GUI应用)
- Flask (用于Web应用)

## 使用说明

每个项目目录都包含独立的代码和说明，可以根据需要单独运行。大多数项目需要先准备相应的数据集，然后运行训练脚本，最后使用测试或可视化脚本查看结果。

## 注意事项

1. 部分项目需要预先下载数据集，如MNIST数据集
2. 训练模型可能需要较长时间，建议使用GPU加速
3. 预训练模型文件可能需要从相应位置下载

## 许可证

本项目仅用于学习和研究目的。
